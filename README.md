# Xendit API

This application contains Xendit's Ride APIs that can be integrated in your application.

## Setup

1. Ensure `node (>8.6 and <=10)` and `npm` are installed.
2. Run `npm install`

## Run the App

`npm start`

## Run the tests

`npm run test`

## Documentation

You can view the full API documentation by accessing `{basePath}/docs` on your browser after running the app.
